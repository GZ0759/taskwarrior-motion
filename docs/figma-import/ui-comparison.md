# UI 实现 vs 设计稿对比

> 给设计师的参考文档。基于当前代码实现整理。

---

## 整体布局

### 设计稿
- 双栏布局：左侧面板（310px）+ 右侧面板（flex-1）
- Mesh 渐变背景

### 实现
- ✅ 与设计稿一致
- 左侧面板新增：热力图 + 项目/标签 tab 切换
- 右侧面板新增：事项/看板/日历 三个 tab 切换

---

## 左侧面板

### 设计稿
- Logo + 音效/主题按钮
- 热力图（35天）
- 项目进度条

### 实现（新增内容）
- ✅ Logo + 音效/主题按钮（主题支持 light/dark/system 三态）
- ✅ 热力图（35天）— **新增交互：点击日期弹窗显示当日完成任务**
- ✅ 项目进度条 — **新增交互：点击项目名弹窗管理（重命名/删除/新建）**
- ✅ **新增标签 tab** — 显示前 8 个高频标签 + 使用次数，点击弹窗管理

---

## 右侧面板

### 设计稿
- 标题 "待办事项"
- 添加任务输入框
- 任务卡片列表
- 空状态

### 实现（变更内容）
- ✅ 标题跟随 tab 变化（事项/看板/日历）
- ✅ 副标题显示动态统计
- ✅ 添加任务输入框（玻璃拟态风格）
- ✅ 任务卡片 — **改为两行布局**：
  - 第一行：checkbox + 描述 + 截止日期
  - 第二行：标签 pills + 优先级 badge + 计时按钮 + 编辑按钮
- ✅ **新增已完成折叠区域**（列表底部，点击展开）
- ✅ 看板视图（5列：Inbox/Backlog/InProgress/OnHold/Done）
- ✅ 日历视图（月/周切换）

---

## 弹窗组件

### 设计稿
- CompletionModal（完成弹窗 + 粒子效果）

### 实现（新增弹窗）
- ✅ CompletionModal — 与设计稿一致
- ✅ **TaskEditModal** — 编辑任务弹窗（优先级/截止日期/等待日期/项目/标签）
- ✅ **DayCompletedModal** — 热力图点击弹窗（显示当日完成任务）
- ✅ **ProjectManageModal** — 项目管理弹窗（重命名/删除/任务列表）
- ✅ **TagManageModal** — 标签管理弹窗（重命名/删除/任务列表）
- ✅ **HelpModal** — 键盘快捷键帮助弹窗

---

## 交互变更

| 功能 | 设计稿 | 实现 |
|------|--------|------|
| 编辑任务 | 展开卡片内联编辑 | 弹窗编辑（所有视图统一） |
| 完成任务 | shake → check-pop → 淡出 → CompletionModal | ✅ 一致 |
| 热力图 | 静态显示 | 可点击查看当日完成任务 |
| 项目管理 | 无 | 点击项目名打开管理弹窗 |
| 标签管理 | 无 | 左侧 tab 切换 + 点击标签打开管理弹窗 |
| 看板 | 按钮切换状态 | ✅ 一致 + 新增"暂停"按钮 |
| 键盘快捷键 | 无 | j/k/x/n/Enter/Escape/? |
| 主题 | light/dark | light/dark/system（SunMoon 图标） |

---

## 视觉风格

全部遵循设计稿的玻璃拟态规范：
- 半透明背景 + backdrop-blur
- Mesh 渐变背景（紫/蓝/粉/橙）
- 大圆角（面板 24px，卡片 16px）
- 项目配色卡片渐变
- 优先级颜色：H 红/M 黄/L 蓝
