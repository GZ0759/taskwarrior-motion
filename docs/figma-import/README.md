# Figma 导入资料

> 本文件夹包含将当前代码实现导入 Figma 所需的所有资料。

## 文件说明

| 文件 | 用途 |
|------|------|
| `prompt.md` | 发给 Figma Make AI 的提示词 |
| `design-tokens.md` | 所有精确视觉数值（颜色/渐变/圆角/间距/动画） |
| `ui-comparison.md` | 原始设计稿 vs 当前实现的对比文档 |
| `screenshots/` | 各视图截图（待补充） |

## 使用方式

1. 打开浏览器，访问 http://localhost:5173
2. 按下方清单截图，保存到 `screenshots/` 文件夹
3. 打开 Figma Make
4. 将 `prompt.md` 内容复制粘贴给 Figma Make
5. 将 `screenshots/` 下的所有图片 + `design-tokens.md` 作为附件上传
