## M1：通用组件 — ModalShell + ToggleSwitch + EmptyDayIcon

**目标**：创建 3 个可在多处以 `<Transition>` 复用的一般性组件。
**前置**：M0
**参考**：
- `ModalShell`：mockup App.tsx:286-319
- `ToggleSwitch`：mockup App.tsx:923-933
- `EmptyDayIcon`：mockup App.tsx:191-207

### 1a. ModalShell.vue

**Props**：
- `title: string` — 标题文字
- `subtitle?: string` — 副标题（可选）
- `isDark: boolean` — 主题
- `maxW?: string` — CSS max-width（默认 `max-w-md`）

**Emits**：`close`

**实现要点**：
1. 外容器 `fixed inset-0 z-50 flex items-center justify-center p-6`
2. `<div class="absolute inset-0" class="glass-overlay" @click="$emit('close')" />`
3. `<Transition name="modal">` + `v-show="true"`（父组件控制显隐）
4. 卡片容器：`rounded-3xl flex flex-col`，`max-h-[80vh]`
5. 暗色背景：`rgba(12,6,26,0.62)` + `blur(52px) saturate(220%)` + 内嵌 1.5px 白色 + 阴影 `0 50px 100px rgba(0,0,0,0.65)`
6. 亮色背景：`rgba(252,250,255,0.74)` + `blur(52px) saturate(180%)` + 内嵌 1.5px 白色 + 阴影 `0 32px 80px rgba(80,60,180,0.12)`
7. Header：`flex items-start justify-between px-6 py-5 shrink-0` + 下划线分隔
8. Body：`flex-1 overflow-y-auto px-6 py-5 space-y-4`

**样式**：使用内联 `:style` 绑定，不依赖 CSS 类

**验收**：
- `modal-shell.spec.ts`：渲染 title、subtitle、slot，emit close
- 手动：`<ModalShell title="测试" :isDark="true" @close="..."><p>内容</p></ModalShell>` 显示正确

### 1b. ToggleSwitch.vue

**Props**：`on: boolean`、`onChange: (v: boolean) => void`

**实现要点**：
1. `<button>` `w-10 h-6 rounded-full relative`，`:style="{ background: on ? '#6366F1' : 'rgba(255,255,255,0.14)' }"`
2. 圆形滑块：`absolute top-0.5 w-5 h-5 rounded-full bg-white`，`x` 动画 `on ? 18 : 2`

**验收**：
- 点击触发 `onChange(!on)`
- 视觉：开启=#6366F1，关闭=semi-transparent white

### 1c. EmptyDayIcon.vue

**Props**：`isDark: boolean`

**实现要点**：
1. SVG 68x68：外层虚线圆环（stroke-dasharray 5 3.5）+ 月牙 + 4 颗星星
2. 颜色用 `isDark ? 'rgba(255,255,255,0.22)' : 'rgba(15,10,40,0.16)'`

**验收**：渲染 68x68 SVG，dark/light 颜色不同

### 测试

新增 `client/tests/components/ModalShell.spec.ts`、`ToggleSwitch.spec.ts`、`EmptyDayIcon.spec.ts`，测试渲染和交互。
