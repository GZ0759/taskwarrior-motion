# 设计导出

> 本文件夹包含将当前代码实现反馈给 Figma 设计师所需的所有文件。

## 文件说明

| 文件 | 用途 |
|------|------|
| `prompt.md` | 发给设计师或 Figma AI 的提示词 |
| `ui-comparison.md` | 设计稿 vs 实现的对比文档 |
| `../design-tokens.md` | 所有精确视觉数值（颜色/间距/圆角等） |

## 使用方式

1. 打开浏览器，访问 http://localhost:5173
2. 截取各视图的截图（暗色/亮色两种主题）
3. 将 `prompt.md` 内容复制发给设计师
4. 将截图 + `ui-comparison.md` + `design-tokens.md` 作为附件

## 需要截图的关键状态

- [ ] 事项视图（暗色 + 亮色）
- [ ] 事项视图 — 已完成展开状态
- [ ] 看板视图
- [ ] 日历视图（月）
- [ ] 日历视图（周）
- [ ] 编辑弹窗

- [ ] 项目管理弹窗
- [ ] 标签管理弹窗
