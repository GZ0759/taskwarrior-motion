## 需要设计的页面（共 10 个）

### 页面 1：事项视图 — 暗色模式
- 左侧：热力图 + 项目进度 tab
- 右侧：任务卡片列表（两行布局）+ 底部「已完成 (N)」折叠

### 页面 2：事项视图 — 亮色模式
- 同页面 1，但使用亮色主题

### 页面 3：看板视图
- 5 列：Inbox / Backlog / In Progress / On Hold / Done
- 每列内卡片有操作按钮（开始/完成/暂停/编辑）

### 页面 4：日历视图 — 月
- 7×6 网格，今日高亮，任务 chips 按优先级着色

### 页面 5：日历视图 — 周
- 7 列，每列显示一天的任务

### 页面 6：编辑弹窗（TaskEditModal）
- 玻璃拟态弹窗
- 字段：优先级（H/M/L 按钮组）、截止日期、暂停到、项目选择器、标签选择器
- 底部：取消/保存/删除

### 页面 7：热力图点击弹窗（DayCompletedModal）
- 显示日期标题 + 完成任务数
- 任务列表：实心圆打勾 + 描述 + 项目 + 完成时间

### 页面 8：项目管理弹窗（ProjectManageModal）
- 头部：项目名 + 进度条 + 统计
- 操作：重命名/删除
- 任务列表：待办 + 已完成分组
- 底部：新建项目名

### 页面 9：标签管理弹窗（TagManageModal）
- 头部：标签名 + 任务数
- 操作：重命名/删除
- 任务列表：待办 + 已完成分组

### 页面 10：完成弹窗（CompletionModal）
- 橙→粉渐变卡片
- SVG 成就徽章 + 粒子爆炸 + 浮动星星
- 统计：今日完成 + 累计完成
- 「继续加油」按钮

**第二行**：

- 计时按钮 + 计时器
- 编辑箭头

## 视觉规范

### 玻璃拟态
- 面板：`backdrop-filter: blur(24px)` + 半透明背景
- Dark：`rgba(255,255,255,0.07)` + `border rgba(255,255,255,0.13)`
- Light：`rgba(255,255,255,0.55)` + `border rgba(255,255,255,0.80)`

### 圆角
- 面板：24px
- 卡片/弹窗：16px
- 按钮：12px
- 标签 pill：9999px

### Mesh 渐变
- Dark 底色：`#060010`
- Light 底色：`#EBE8FF`
- 5 层 radial-gradient（紫/蓝/粉/青/橙）

### 项目配色
- Design System：紫色渐变
- API Migration：蓝色渐变
- Marketing：粉色渐变
- Q3 Planning：橙色渐变
